UpLove Slash Command Overview
=============================

- `/help overview` — category summary and quick links.
- `/help category <name>` — drill into a command family.
- `/help admin` — detailed administrator playbook.
- `/relaystatus`, `/relayping`, `/relayshutdown` — relay operations.
- `/coinflip`, `/roll`, `/pick`, `/slots`, `/gamble`, `/hangman start`, `/tictactoe start`, `/wordladder`, `/trivia`, `/credits`, `/reward` — games, rewards, and community fun.
- `/monitor list|add|remove` — manage monitoring targets.
- `/rss list|add|remove` — manage RSS feeds.
- `/relayreload`, `/relayrestart`, `/relayannounce`, `/relaystats`, `/relaydebug` — admin controls.
- `/kick`, `/ban`, `/unban`, `/mute`, `/timeout`, `/roleadd`, `/roleremove`, `/temprole` — moderation toolkit.
- `/znc config` — ZNC server configuration.
